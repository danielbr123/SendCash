# SendCash
Economy plugin for PocketMine-MP.OPからオンラインのプレイヤーに一斉に送金できたりするプラグイン。OPの所持金は減りません。
##How to use
/cash send <プレイヤー> <金額>...<player>に<金額>を送金します。
/cash grant <金額>...<金額>を送金します。
